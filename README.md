# 🚀 TechPulse

TechPulse is an AI-powered career intelligence platform designed to track technology evolution, analyze job market demand, and provide actionable insights for developers. 

By aggregating market signals and leveraging LLM-based synthesis, the platform delivers real-time strategic intelligence reports for any technology, framework, or industry trend.

## 🛠️ Tech Stack

**Frontend (Client)**
* React 19 + Vite
* Tailwind CSS + Framer Motion (Animations)
* Lucide React (Icons)
* Recharts (Data Visualization)

**Backend (API & Orchestration)**
* Node.js + Express
* Prisma ORM (SQLite for dev, easily migrateable to PostgreSQL)
* Google GenAI / OpenAI APIs (Intelligence Synthesis)
* JSON Web Tokens (JWT) for Authentication
* Node-Cron (Automated market data fetching)
